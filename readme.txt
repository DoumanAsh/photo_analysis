Короткая справка по программе "Менеджер фотопроектов"
====================================================

Программа предназначена для организации хранения фотоснимков в виде фотопроектов.
Фотопроект - ограниченный по времени и теме набор снимков.

=====================
Выбор проектной папки
---------------------
В главном окне: меню Проекты -> Настройки -> кнопка с путём к текущей проектной папке

=====================
Создание фотопроектов
---------------------
В главном окне: меню Проекты -> Создать проект
В окне редактирования проектов: занести данные в форму -> кнопка Сохранить

==============================
Добавление фотографий в проект
------------------------------
В главном окне: меню Общие команды -> Сохранить фотографии -> Выбрать папку, откуда необходимо сохранить фотографии
Фотографии из выбранной папки будут автоматически распределны по проектам на основе даты и времени.
Если проект для фотографии не найден, она будет сохранена в папку _unsorted.
Фотографии из папки _unsorted могут быть сохранены автоматически при следующем сохранении в зависимости от настроек программы.

===========================
Анализ фотографий в проекте
---------------------------
В главном окне в проектном режиме: кнопка Анализ фото

=========================
Анализ фотографий в папке
-------------------------
В главном окне: меню Общие команды -> Выполнить анализ фотографий

=============================
Выполнение анализа фотографий
-----------------------------
В окне анализа фотографий: кнопка Анализировать
Для сохранения информации в метаданные фотографии после анализа: кнопка Сохранить

=====================
Просмотр фотопроектов
---------------------
В главном окне: меню Проекты -> Обзор проектов

Для выбора проекта: двойной клик ЛКМ на проекте или выделить проект -> клавиша Enter
Для удаления проекта: выделить проект -> клавиша Delete

===========================
Редактирование фотопроектов
---------------------------
В главном окне в проектном режиме: кнопка Изменить проект

========================================
Выход из проектного режима главного окна
----------------------------------------
В главном окне в проектном режиме: кнопка Закрыть проект или меню Проекты -> Закрыть проект
